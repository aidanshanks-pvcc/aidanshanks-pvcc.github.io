Operating Systems Terminology
Term Definition
Operating SystemThe software that controls the overall operation of a computer
Job QueueA storage organization method in which objects are ordered first in and last out
FIFOAn abrreviation to "First in, Last out" where jobs are completed in the order they arrive
Interactive ProcessingA way to allow users to modify programs on a computer through the use of a remote terminal
Time-SharingA way to assist multiple users at the same time, even if they are attempting to perform different tasks
MultiprogrammingA manner of time sharing in which the computer rapidly shifts between jobs in order to appear as though they finish simultaneously
Application softwarePrograms for performing taks particular to the machine's specialization
System SoftwarePrograms that perform common tasks on computers
Utility SoftwarePrograms that are not inherent in an operating system but still perform fundamental tasks
Graphical User Interface(GUI)A method of allowing users to perform actions through the use of pictures instead of words
ShellA method of operating a computer comprising solely of text
KernelThe most basic components of a computer's operating system
Device DriverThe software components that communicate with the controllers or other such peripheral devices
File ManagerMaintains records of all the files stored in mass storage
Memory ManagerCoordinates the usage of the machine's use of its main memory
Virtual MemoryA way to temporarily utilize more space than allocated in memory by shifting between "pages" that contain the necessary programming to run whatever you are working on
Boot StrappingThe initial wakeup programming in a computer. Runs on launch
ROMRead only memory:contains information intended to e read but not altered. The programming for a boot strapping is stored in here
DeadlockA condition in which 2 or more processes cannot occur because each is awaiting a resource allocated to the other
SpoolingA method of preventing deadlock by disassociating the accomplishment of each task with the ability to run future tasks
Super User(Or Administrator)A method of preventing hacking by providing one user a higher level of access to modify or access the programs on the computer
Multi-factor AutdenticationAnother way to prevent hacking by requiring the use of more account to verify the identity of the person attempting to access something on the computer/the internet
