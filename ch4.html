Operating Systems Terminology
Term Definition
Network Classifications
LANLocal Area Network. A group of computers in a small area
MANMetropolitan Area Network. A group of computers in a single city
WANWide Area Network. A group of computers between neighboring cities or wider
Network Models
Bus NetworkA system of connecting computers in which all computers are connected to a common communication line, or bus
Star NetworkA system of connecting computers where all computers are connected to a central focus point
Protocols
CSMA/CD EthernetCarrier Sense, Multiple Access with Collision Detection. Each machine cpaces out their sending time and read all messages on the line, only keeping the ones addressed to them
CSMA/CA Wifi"", with collision avoidance. A system that tries to avoid collisions and resends a message when a collision occurs. Typically gives priority in a job queue
Combining Networks
RepeaterA device that passes signal back and forth between 2 buses
BridgeSimilar to a repeater, but only passes along a message to a designated recipient
SwitchA bridge that has multiple connections, allowing for more buses to be connected
RouterA special computer that is used to communicate with other routers connected to other netwroks, allowing for cross-network communication
GatewayThe thing a network uses to communicate with other networks. Often the router
Client/Server Model
P2P:Peer-to-peerA more temporary method of transmitting data by sending it directly from one device to another
Distributed systemA set of software consisting of individual pieces of software that execute as processes on different computers
Cloud ComputingThe outsourcing of large-scale computing to a set of computers whose sole task is to compute those computations
Internet Addressing
Tier 1The backbone of the internet. Composed of very powerful and eficient computers operating in a WAN
Tier 2The ISPs connected to the tier 1 ISPs which operate regionally
Tier 3The LAN that runs off of the tier 1 and 2 ISPs
IP address
ICANNInternet corporation for assigned names and numbers.A nonprofit corporation that coordinates the internet's operation
DNSDomain Name System. a system in which each "region"(think websites) on the internet is assigned a domain name, a unique identifier for that "region" of the internet
TLD:Top Level DomainTLDs are the basic suffixes attached to the back of a website's url. This includesw .com .edu and .gov
Country-code TLDsThese are TLDs that are related to a country such as .au for Australia that can be used to subdivide a site between regions. This allows for the provision of different content based on varying laws between countries. A list of Country's TLDs.
Internet Protocols
FTPFile Transfer Protocol. An application for accessing another computer from a long distance.
HTTPHyper Text Transfer Protocol. A powerful and secure way to transport text data between computers. Commonly used for websites
SMTPSimple Mail Transfer Protocol. Defines how computers may ineract when sending emails between computers.
World Wide Web
URLUniform Resource Locator. Contains the information needed by a browser to connect to a server and access the desired text file
client-side activitiesActivities performed by a client(like a browser)
server-side activitiesActivities performed by a server(like a webserver)
Internet four-layer software
Application LayerPrepares messages and provides destination addresses
Transport LayerSplits messages into packets
Network LayerAssigns an intermediate address to each packet
Link LayerTransfers the packets
Internet is:
Packet-switching NetworkThe internet works by handing off small chunks of information(called packets) between computers
Fault TolerantAble to respond to errors and keep shifting information smoothly
RedundantHas multiple protocols that do similar effects in order to ensure the internet is fault-tolerant
Cybersecurity
MalwareThe collective name for malicious software. comes in many varieties
VirusInserts itself into files already on the computer, executing itself when the host file is executed
WormA hidden program that copies itself and sends it to other computers on the network
Trojan HorseAppears to be a useful file, but instead harms the computer or steals data when the file is run
SpywareSoftware that collects sensitive info from a user, typically for a nefarious purpose.
PhishingA manner of collecting data by asking someone for it, often under false pretenses. Scams are a type of phishing.
DDoS AttackDenial of Service. A way to prevent a computer from accessng anything by overloading a computer with messages.
Protection
FirewallA filter that scans programs to ensure they will not cause harm to the computer
Spam FilterRemoves extraneous emails from untrustworthy sources in order to reduce the number of phishing attempts that suceed
Proxy ServerA software unit that acts as an intermediary in order to provide a meat shield to reduce the number of secrets stolen from the important information
Antivirus SoftwareA way to detect and remove viruses and other computer infections
CryptographyA way to encrypt messages by transmuting important information into a different strings of characters that can be translated by the intended receiver.
Public Key EncryptionA two step sytem of encrypting information by requiring one key to send and a different key to receive the information.
